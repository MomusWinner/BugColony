# Description
Bug Colony is a 3D simulation of a bug colony. Bugs move around the scene, collect resources, reproduce and mutate.

# Installation
1. Load project from repositorty `git clone https://github.com/MomusWinner/BugColony`.
2. Restore NuGet packages via NuGet > Restore Packages.

# Stack
 - VContainer
 - R3
 - UniTask
 - MessagePipe
 - Tri Inspector
